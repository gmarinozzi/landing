SER-VI-CIOS

  • Web Apps
    Son sitios web que permiten la interacción con usuarios y redes sociales. Nuestros diseños son perfectamente adaptables a tablets y smartphones y nuestro código basado en un API central permite crear aplicaciones móbiles de forma muy eficiente.
  • Mobile Apps
    Son aplicaciones especialmente creadas para interactuar con dispositivos móviles y tabletas. Nuestra estructura de trabajo basada en un API central, nos permite crear aplicaciones móbiles a partir de una aplicación Web de forma muy sencilla.
  • Consultoría y capacitación
    Contamos con la experiencia del trabajo y la capcitación constante. Vemos en cada consultoría un reto para aprender nuevas tecnologías, y en cada capacitación la oportunidad compartirlas para revivir la pasión de haberlas aprendido.
